# Deployment (TBD)

Our Starter Environment and RV Starter Theme is hosted at WPEngine. To understand more about the WPEngine deployment setup, please refer to the [GitHub Action for WP Engine Site Deployments](https://wpengine.com/support/github-action-deploy/) documentation.

## Test environment

Test environment is the only environment we have for the RV Starter Theme. It uses the [wpengine-deploy-test.yml](../../.github/workflows/wpengine-deploy-development.yml) deployment file.

In order to deploy you can either:

- push changes to the `master` branch.
- merge a PR to the `master` branch.
- run the "Deploy to the WP Engine Test environment" on GitHub action.

Once deployed, the changes will be available at TBD.
